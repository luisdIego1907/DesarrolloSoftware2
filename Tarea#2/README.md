Qué es una hilera de caracteres y algunos ejemplos:

    -Una hilera de caracteres viene a ser una secuecnia ordenada de caracteres, que puede ser letras,
    números, símbolos o espacios.
    -El término para referese a una hilera es String.
    -Un String es un objeto que representa texto.
    -En java los String son inmbutables.

Diferencia entre un carácter y una hilera:

    Caracter:
        -Un carácter representa un solo símbolo.
        -Es de tipo primitivo
        -Se escribe con comillas simples
        -Se almace un único caracter
    
    Hilera:

        -Secuencia de caracteres.
        -Es un objeto.
        -Se escribe con comillas dobles.
        -Puede contener uno o muchos caracteres
        -Es inmutable.

Operaciones que se pueden realizar con una hilera:

    Concatenar, obtener la longitud, acceser a un caracter específico,
    comparar hileras, convertiras a mayúsculas o minúsculas, extraer una subparte,
    reemplazar caraccteres o palabras,
